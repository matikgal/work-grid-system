# Work Grid System

Nowoczesne narzędzie typu "Operations Hub" do kompleksowego zarządzania grafikami pracy, planowania zmian oraz ewidencji czasu pracy zespołu. Aplikacja łączy w sobie funkcjonalność kalendarza, systemu raportowania oraz bazy danych pracowników, opakowaną w intuicyjny i estetyczny interfejs.

## Demo

Zobacz aplikację na żywo: <a href="https://matikgal.github.io/work-grid-system/" target="_blank">Link</a>

## O projekcie

Projekt Work Grid System powstał w celu usprawnienia procesów planowania w zespołach, eliminując konieczność korzystania z przestarzałych arkuszy kalkulacyjnych. Głównym celem było stworzenie "centrum dowodzenia", które pozwala na szybkie tworzenie grafików, zarządzanie urlopami oraz automatyczne zliczanie godzin pracy. System wyróżnia się ciemnym, nowoczesnym interfejsem (Dark Mode) oraz wysoką responsywnością, zapewniając komfort pracy zarówno na desktopie, jak i urządzeniach mobilnych.

## Główne funkcjonalności

- **Interaktywny Kalendarz:** Intuicyjny grid do planowania zmian z obsługą "Drag & Drop" oraz szybkim dodawaniem.
- **Obsługa Zmian i Absencji:** Predefiniowane typy zmian (6-14, 14-22) oraz pełna obsługa urlopów, L4 i dni wolnych.
- **System Raportowania:** Automatyczne generowanie raportów miesięcznych gotowych do druku (Print View).
- **Zarządzanie Zespołem:** Baza danych pracowników z historią zmian i statystykami godzin.
- **Autentykacja:** Bezpieczne logowanie i zarządzanie sesją użytkownika oparte o Supabase.
- **Responsywność:** Aplikacja dostosowana do działania na różnych rozdzielczościach ekranu.
